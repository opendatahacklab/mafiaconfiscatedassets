INTRODUCTION

In this project we develop some tools and mantain knowledge bases related to confiscated assets. The goal is to build a huge knowledge base of confiscated assets in Sicily: a generic vocabulary to represent confiscated assets, an ontology of which public administrations provide enough information about confiscated assets it owns, ontologies of these assets. We release the tools used to build this knowledge base as well, in particular tools to convert assets information from its origin format (may be, for example, CSV) to the uniform format we provide.

DATA

Some data are produced and distributed in the scope of this project.

An OWL vocabulary (confiscatedassets_vocab.owl) to define classes and properties has been created and it is currently work in progress.

An catalogue of datasets concerning confiscated assets is available as well. These can be accessed as indicated in the open data page of this project. In addition, you can browse the catalogue by accessing the datasets page.

ONTOLOGIES and Maps

Aside the, some of these datasets listed in the confiscated assets datasets ontology have been converted into a uniform notation based on semantic web technologies, using the confiscated assets vocabolary.

In particular, for some of the datasets released by municipalities, a wrapper script has been provided. Such a wrapper operates the convertion from the original format in which the dataset is released into the one which uses our confiscated assets vocabulary.

For each municipality, all the related stuffs are available in the gitHub repository of this project in a directory named with the municipality name. In addition, instruction for online access to the produced ontologies are provided in the ontologies section of the open data page.
