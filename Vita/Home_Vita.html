The ontology of confiscated assets in Vita has been produced from the Elenco Beni Confiscati dataset released by the municipality of Vita. The file is in PDF format. Thus we had to extract the table, converting them in CSV, by the extraction tool Tabula. Then, this file has been manually updated to solve some issues (for example, keep separate address and civic number). Finally, it has been converted in RDF by the Generate_File_RDF_XML.php script and inferences have been performed using the reasoner Pellet. The ontology is available as OWL downloadable files:

  • assets.owl
  • locations.owl
  • geometry.owl

In addition, it can be accessed via its own SPARQL Endpoint.

USAGE NOTE: coordinates reported in geometry.owl and accessible via the SPARLQ Endpoint have been obtained by the google geocoder. As consequence, this information is not freely reusable, but it is subject to the conditions imposed by google.
